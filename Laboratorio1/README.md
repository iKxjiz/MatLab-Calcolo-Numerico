## Argomento

Primo contatto con l'ambiente **MATLAB/Octave**: il "laboratorio virtuale" che useremo per tutto il corso.

**Concetti chiave:**

- **Workspace e Command Window** — dove vivono le variabili e dove si impartiscono comandi
- **Script `.m`** — file di testo con sequenze di istruzioni MATLAB, eseguibili come un unico blocco
- **Built-in functions** — funzioni già pronte (es. `sin`, `cos`, `sqrt`, `plot`...)

**Perché è importante:**  
MATLAB sarà lo strumento con cui sperimenteremo tutti i metodi numerici del corso: dalla rappresentazione dei numeri finiti, alle curve di Bézier, fino ai sistemi lineari.

## Tips

- Se usi **Octave** (alternativa gratuita a MATLAB), la sintassi è praticamente identica

- Salva spesso i tuoi script con nomi significativi!
